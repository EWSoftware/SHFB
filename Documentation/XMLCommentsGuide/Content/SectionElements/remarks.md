This element is used to provide a more detailed description of the type or member.



## Syntax

This top-level element is valid on all types and type members.


``` xml{title=" "}
<remarks>description</remarks>
```


## Remarks

Use it when necessary to provide more detailed information about the type or member such as how it
is used, its processing, etc.  Remarks will only appear in the help file and can be as long as necessary.
