This element is used to provide a brief description of the type or member.



## Syntax

This top-level element is valid on all types and type members.


``` xml{title=" "}
<summary>description</summary>
```


## Remarks

The description will appear in IntelliSense and the Object Browser.  As such, it should be kept
brief.  Use the `remarks` element to provide more detailed information about the type or
member.
