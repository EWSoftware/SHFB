This element is used to indicate that text within a description should be marked as code.



## Syntax

This inline element can be used within any other element.


``` xml{title=" "}
<c>text</c>
```


## Remarks

Use this element to mark inline text as code.  Use the [](@1abd1992-e3d0-45b4-b43d-91fcfc5e5574)
element to mark a multi-line block of text as code.
