This element is used to start a new paragraph within other elements.



## Syntax

This inline block element can be used within any other element.


``` xml{title=" "}
<para>paragraph content</para>
```


## Remarks

As an alternative, you can use the HTML &lt;p /&gt; or &lt;p&gt; &lt;/p&gt; tags to create
paragraphs.  Making the element self-closing or empty will cause the element to be ignored unless it creates an
initial paragraph break.
