Changes made in this release:

- Fixed a null reference exception in the standard document model generator.
- Fixed target framework identifier handling so that it worked correctly with `net10.0` targets.
- Fixed handling of `code` elements with no attributes so that they are rendered as blocks rather than inline
  elements if nested within an expected block-level parent element to match legacy behavior.
